## Changes From Design Document
- Background music was not included
- Fewer than 15 rooms were created for a MVP
- No exposition was included beyond initial mini cutscene
- Bats were used as a substitute for wraiths
- Items were changed to being just collectable enemy drops for restoring mana and snowpity
- Shooting uses arrow keys instead of space bar
- No 2.5D done with columns
- Used nether portals as the system for moving between rooms
